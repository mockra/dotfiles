# README

## Getting Started

### Clone Repo

`git clone git@github.com:mockra/dotfiles.git .`

### Homebrew

Visit [homebrew website](https://brew.sh/) for install commands

Run: `brew bundle`

### Monolith Git Config
```
watchman watch-project ~/my-mono-repo
cd ~/my-mono-repo
git config core.fsmonitor rs-git-fsmonitor
```
